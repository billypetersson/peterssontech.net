About

I'm a Microsoft Certified Azure Engineer at Innofactor, where I design, build and manage cloud infrastructure — mostly through Azure DevOps and Bicep.

My path into the cloud ran through years of hands-on IT consulting and infrastructure work across Örebro and Stockholm. That foundation — real systems, real users, real uptime — shapes how I approach the cloud today: infrastructure as code, repeatable pipelines, and security baked in from the start rather than bolted on later.

I care about clean, reviewable infrastructure that a team can trust and reason about — the kind you can read like a manifest and deploy with confidence.
